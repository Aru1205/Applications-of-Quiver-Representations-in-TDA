
This repository contains my SURGE reading project completed under **Prof. Amit Kuber** at **IIT Kanpur**.

## Overview

The project explores how **quiver representations** provide an algebraic framework for studying **persistent homology** in Topological Data Analysis (TDA). It develops the mathematical foundations required to understand persistence modules and illustrates their application through worked examples.

## Topics Covered

- Quiver representations
- Persistence modules
- Simplicial complexes
- Simplicial homology
- Chain complexes and boundary maps
- Persistence diagrams
- Applications of Topological Data Analysis
